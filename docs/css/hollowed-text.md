## 镂空文字

在字体颜色设置中只能设置一些单调的颜色，一点都不炫酷，在css3当中我们可以通过设置[background-clip](https://developer.mozilla.org/zh-CN/docs/Web/CSS/background-clip)，将背景被裁剪成文字的前景色，来实现一些有趣的效果。
